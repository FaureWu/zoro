插件介绍

插件

自zoro2.*版本之后，已经将插件部分独立出来，仓库地址zoro plugin

如何安装zoro插件请查看安装文档

内置插件

  • loading插件 用于全局管理loading状态，详情查看LOADING插件
  • mixin插件 提取model公共逻辑，详情查看MIXIN插件
  • perf插件 收集查看相关性能数据，用于辅助性能优化，详情查看PERF插件
