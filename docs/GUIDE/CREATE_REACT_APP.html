接入create-react-app

CREAT_REACT_APP中使用

create-react-app脚手架的搭建使用，请移步create-react-app

成功创建项目后除了参照安装文档安装zoro以外，我们还需要安装如下依赖

$ npm install --save react-redux

或者使用yarn管理依赖

$ yarn add react-redux

参照快速开始
